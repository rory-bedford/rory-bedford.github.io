Problem Specification

\[ \begin{align} y_i &= \frac{1}{\sqrt{n}}\sum_{j=1}^nw_{ij}x_j\\ \hat{\mu} &= \frac{1}{n}\sum_{i=1}^ny_i\\ \hat{\sigma}^2 &= \frac{1}{n}\sum_{i=1}^n (y_i-\hat{\mu})^2 \end{align} \]

\[ \begin{align} w_{ij} &\overset{iid}{\sim} \mathbb{P}(w)\\ x_{j} &\overset{iid}{\sim} \mathbb{P}(x) \end{align} \]

Find: \[ \begin{align} \mathbb{E}[\hat{\mu}]\text{, } & \text{Var}(\hat{\mu})\\ \mathbb{E}[\hat{\sigma}^2]\text{, } & \text{Var}(\hat{\sigma}^2) \end{align} \]

Some Notation

\[ \begin{align} \mu_x &= \mathbb{E}[x_j]\\ \mu_w &= \mathbb{E}[w_{ij}]\\ \sigma_x^2 &= \text{Var}(x_j)\\ \sigma_w^2 &= \text{Var}(w_{ij}) \end{align} \]

\[ \begin{align} y_i &= \frac{1}{\sqrt{n}}\sum_{j=1}^nw_{ij}x_j\\ \hat{\mu} &= \frac{1}{n}\sum_{i=1}^ny_i\\ \hat{\sigma}^2 &= \frac{1}{n}\sum_{i=1}^n (y_i-\hat{\mu})^2 \end{align} \]

Crucial Observation

For \(i \neq j\): \[ y_i \not\!\perp\!\!\!\perp y_j \] Through joint dependence on the same samples \(\{x_j\}_{j=1}^n\)

\[ \begin{align} y_i &= \frac{1}{\sqrt{n}}\sum_{j=1}^nw_{ij}x_j\\ \hat{\mu} &= \frac{1}{n}\sum_{i=1}^ny_i\\ \hat{\sigma}^2 &= \frac{1}{n}\sum_{i=1}^n (y_i-\hat{\mu})^2 \end{align} \]

Crucial Observation

\[ \begin{align} y_i &= \frac{1}{\sqrt{n}}\sum_{j=1}^nw_{ij}x_j\\ \hat{\mu} &= \frac{1}{n}\sum_{i=1}^ny_i\\ \hat{\sigma}^2 &= \frac{1}{n}\sum_{i=1}^n (y_i-\hat{\mu})^2 \end{align} \]

Approach

Compute moments of \(y_i\) first, including covariances, then use the central limit theorem since \(y_i\) is the sum of a large number of i.i.d. variables.

\[ \begin{align} y_i &= \frac{1}{\sqrt{n}}\sum_{j=1}^nw_{ij}x_j\\ \hat{\mu} &= \frac{1}{n}\sum_{i=1}^ny_i\\ \hat{\sigma}^2 &= \frac{1}{n}\sum_{i=1}^n (y_i-\hat{\mu})^2 \end{align} \]

Mean:


\(\mu_y=\mathbb{E}[y_i]\)

\(\mu_y = \mathbb{E}\left[\frac{1}{\sqrt{n}}\sum_{j=1}^nw_{ij}x_j\right]\)

\(\mu_y = \frac{1}{\sqrt{n}}\sum_{j=1}^n\mathbb{E}[w_{ij}x_j] \quad\quad \text{Linearity of expectation}\)

\(\mu_y = \frac{1}{\sqrt{n}}\sum_{j=1}^n\mathbb{E}[w_{ij}]\mathbb{E}[x_j] \quad\quad \text{Independence}\)

\(\mu_y = \frac{1}{\sqrt{n}}\sum_{j=1}^n \mu_w \mu_x\)

\(\mu_y = \sqrt{n} \mu_w \mu_x\)

Mean:


\(\mu_y = \sqrt{n} \mu_w \mu_x\)

Variance:


\(\sigma_y^2=\text{Var}(y_i)\)

\(\sigma_y^2 = \text{Var} \left(\frac{1}{\sqrt{n}} \sum_{j=1}^n w_{ij}x_j\right)\)

\(\sigma_y^2 = \frac{1}{n} \text{Var} \left( \sum_{j=1}^n w_{ij}x_j\right)\)

\(\sigma_y^2 = \text{Var} (w_{ij}x_j) \quad\quad \text{i.i.d.}\)

\(\sigma_y^2 = \mathbb{E}[w_{ij}^2x_j^2] - \mathbb{E}[w_{ij}x_j]^2\)

\(\sigma_y^2 = \mathbb{E}[w_{ij}^2]\mathbb{E}[x_j^2] - \mathbb{E}[w_{ij}]^2\mathbb{E}]x_j]^2 \quad\quad \text{Independence}\)

\(\sigma_y^2 = (\mu_w^2+\sigma_w^2)(\mu_x^2+\sigma_x^2)-\mu_w^2\mu_x^2\)

\(\sigma_y^2 = \sigma_w^2\sigma_x^2 + \sigma_w^2\mu_x^2 + \sigma_x^2\mu_w^2\)

Variance:


\(\sigma_y^2 = \sigma_w^2\sigma_x^2 + \sigma_w^2\mu_x^2 + \sigma_x^2\mu_w^2\)

Covariance \((i \neq j)\):


\(\text{Cov}(y_i, y_j) = \mathbb{E}[y_iy_j] - \mathbb{E}[y_i]\mathbb{E}[y_j]\)

\(\text{Cov}(y_i, y_j) = \mathbb{E}[y_iy_j] - n \mu_w^2 \mu_x^2\)

\(\mathbb{E}[y_iy_j] = \mathbb{E} \left[ \frac{1}{n}\sum_{k=1}^n w_{ik}x_k \sum_{l=1}^n w_{jl}x_l \right]\)

\(\mathbb{E}[y_iy_j] = \frac{1}{n} \mathbb{E} \left[ \sum_{k=1}^n w_{ik}w_{jk}x_k^2 + \sum_{l\neq k}^n w_{ik}w_{jl}x_kx_l \right]\)

\(\mathbb{E}[y_iy_j] = \frac{1}{n} \sum_{k=1}^n \mathbb{E}[w_{ik}]\mathbb{E}[w_{jk}]\mathbb{E}[x_k^2] + \frac{1}{n} \sum_{l\neq k}^n \mathbb{E}[w_{ik}]\mathbb{E}[w_{jl}]\mathbb{E}[x_k]\mathbb{E}[x_l]\)

\(\mathbb{E}[y_iy_j] = \mu_w^2(\mu_x^2+\sigma_x^2) + (n-1) \mu_w^2 \mu_x^2\)

\(\text{Cov}(y_i, y_j) = \mu_w^2\sigma_x^2\)

Covariance \((i \neq j)\):


\(\text{Cov}(y_i, y_j) = \mu_w^2\sigma_x^2\)

Summary So Far

\[ \begin{align} \mathbb{E}[y_i] &= \sqrt{n} \mu_w \mu_x\\ \text{Var}(y_i) &= \sigma_x^2 \sigma_w^2 + \sigma_x^2 \mu_w^2 + \sigma_w^2 \mu_x^2\\ \text{Cov}(y_i,y_j) &= \mu_w^2 \sigma_x^2 \end{align} \]

Normal Distribution:

Going to switch to vector notation:

\(\mathbf{y} = (y_1,...,y_n)^T\)


Since \(y_i\) is a sum of n random variables, by the central limit theorem, as \(n \rightarrow \infty\): \(\mathbf{y} \sim \mathcal{N}(\boldsymbol{\mu}_y, \Sigma_y)\)


\(\boldsymbol{\mu}_y = \sqrt{n}\mu_x\mu_w \boldsymbol{1}\)

\(\Sigma_y = \sigma_w^2 (\sigma_x^2 + \mu_x^2) \mathbb{I} + \mu_w^2\sigma_x^2 \mathbb{J}\)

Rewriting our expressions:


Sample mean:

\(\hat{\mu} = \frac{1}{n}\sum_{i=1}^ny_i\)

\(\hat{\mu} = \frac{1}{n} \boldsymbol{1}^T\mathbf{y}\)


Population variance:

\(\hat{\sigma}^2 = \frac{1}{n}\sum_{i=1}^n (y_i-\hat{\mu})^2\)

\(\hat{\sigma}^2 = \frac{1}{n} \mathbf{y}^T(\mathbb{I} - \frac{1}{n} \mathbb{J})^T(\mathbb{I} - \frac{1}{n} \mathbb{J})\mathbf{y}\)

Simplifying:


\(\hat{\sigma}^2 = \frac{1}{n} \mathbf{y}^T(\mathbb{I} - \frac{1}{n} \mathbb{J})^T(\mathbb{I} - \frac{1}{n} \mathbb{J})\mathbf{y}\)


\((\mathbb{I} - \frac{1}{n} \mathbb{J})^T(\mathbb{I} - \frac{1}{n} \mathbb{J}) = \mathbb{I} - \frac{2}{n} \mathbb{J} + \frac{1}{n^2}\mathbb{J}^2\)

\((\mathbb{I} - \frac{1}{n} \mathbb{J})^T(\mathbb{I} - \frac{1}{n} \mathbb{J}) = \mathbb{I} - \frac{2}{n} \mathbb{J} + \frac{1}{n}\mathbb{J}\)

\((\mathbb{I} - \frac{1}{n} \mathbb{J})^T(\mathbb{I} - \frac{1}{n} \mathbb{J}) = \mathbb{I} - \frac{1}{n} \mathbb{J}\)


\(\hat{\sigma}^2 = \frac{1}{n} \mathbf{y}^T(\mathbb{I} - \frac{1}{n} \mathbb{J})\mathbf{y}\)

\(\mathbb{E}[\hat{\mu}]\):


\(\mathbb{E}[\hat{\mu}] = \mathbb{E}[\frac{1}{n} \boldsymbol{1}^T\mathbf{y}]\)

\(\mathbb{E}[\hat{\mu}] = \frac{1}{n} \boldsymbol{1}^T\mathbb{E}[\mathbf{y}] \quad\quad \text{Linearity expectation}\)

\(\mathbb{E}[\hat{\mu}] = \frac{1}{n} \boldsymbol{1}^T\boldsymbol{\mu}_y\)

\(\mathbb{E}[\hat{\mu}] = \sqrt{n} \mu_x \mu_w\)

\(\mathbb{E}[\hat{\mu}]\):

\(\mathbb{E}[\hat{\mu}] = \sqrt{n} \mu_x \mu_w\)

\(\mathbb{E}[\hat{\sigma}^2]\):


\(\mathbb{E}[\hat{\sigma}^2] = \mathbb{E}[\frac{1}{n} \mathbf{y}^T(\mathbb{I} - \frac{1}{n} \mathbb{J})\mathbf{y}]\)


Matrix cookbook eq. (378):

\(\mathbb{E}[\mathbf{x}^T\mathbf{A}\mathbf{x}] = \text{Tr}[\mathbf{A}\Sigma] + \boldsymbol{\mu}^T\mathbf{A}\boldsymbol{\mu}\)


\(\mathbb{E}[\hat{\sigma}^2] = \text{Tr}((\mathbb{I} - \frac{1}{n} \mathbb{J})\Sigma_y) + \boldsymbol{\mu}_y^T(\mathbb{I} - \frac{1}{n} \mathbb{J})\boldsymbol{\mu}_y\)

\(\mathbb{E}[\hat{\sigma}^2]\):


\(\mathbb{E}[\hat{\sigma}^2] = \mathbb{E}[\frac{1}{n} \mathbf{y}^T(\mathbb{I} - \frac{1}{n} \mathbb{J})\mathbf{y}]\)


Matrix cookbook eq. (378):

\(\mathbb{E}[\mathbf{x}^T\mathbf{A}\mathbf{x}] = \text{Tr}[\mathbf{A}\Sigma] + \boldsymbol{\mu}^T\mathbf{A}\boldsymbol{\mu}\)


\(\mathbb{E}[\hat{\sigma}^2] = \text{Tr}((\mathbb{I} - \frac{1}{n} \mathbb{J})\Sigma_y) + \boldsymbol{\mu}_y^T(\mathbb{I} - \frac{1}{n} \mathbb{J})\boldsymbol{\mu}_y \quad\quad \text{Latter term zero}\)

\(\mathbb{E}[\hat{\sigma}^2]\):


\(\mathbb{E}[\hat{\sigma}^2] = \mathbb{E}[\frac{1}{n} \mathbf{y}^T(\mathbb{I} - \frac{1}{n} \mathbb{J})\mathbf{y}]\)


Matrix cookbook eq. (378):

\(\mathbb{E}[\mathbf{x}^T\mathbf{A}\mathbf{x}] = \text{Tr}[\mathbf{A}\Sigma] + \boldsymbol{\mu}^T\mathbf{A}\boldsymbol{\mu}\)


\(\mathbb{E}[\hat{\sigma}^2] = \frac{1}{n}\text{Tr}((\mathbb{I} - \frac{1}{n} \mathbb{J})\Sigma_y)\)

\(\mathbb{E}[\hat{\sigma}^2] = \frac{1}{n}\text{Tr}((\mathbb{I} - \frac{1}{n} \mathbb{J})(\sigma_w^2 (\sigma_x^2 + \mu_x^2) \mathbb{I} + \mu_w^2\sigma_x^2 \mathbb{J}))\)

\(\mathbb{E}[\hat{\sigma}^2] = \frac{1}{n}\text{Tr}(\sigma_w^2 (\sigma_x^2 + \mu_x^2)\mathbb{I} + \mu_w^2\sigma_x^2 \mathbb{J} - \frac{1}{n}\mu_w^2\sigma_x^2 \mathbb{J}^2) \quad\quad \text{Leading order in n}\)

\(\mathbb{E}[\hat{\sigma}^2] = \frac{1}{n}\text{Tr}(\sigma_w^2 (\sigma_x^2 + \mu_x^2)\mathbb{I})\)

\(\mathbb{E}[\hat{\sigma}^2] = \sigma_w^2 (\sigma_x^2 + \mu_x^2)\)

\(\mathbb{E}[\hat{\sigma}^2]\):

\(\mathbb{E}[\hat{\sigma}^2] = \sigma_w^2 (\sigma_x^2 + \mu_x^2)\)

\(\text{Var}(\hat{\mu})\):


\(\text{Var}(\hat{\mu}) = \text{Var}(\frac{1}{n} \boldsymbol{1}^T\mathbf{y})\)


Matrix cookbook eq. (375):

\(\text{Var}(\mathbf{A}\mathbf{x}) = \mathbf{A}\Sigma\mathbf{A}^T\)


\(\text{Var}(\hat{\mu}) = \frac{1}{n^2} \boldsymbol{1}^T\Sigma_y\boldsymbol{1}\)

\(\text{Var}(\hat{\mu}) = \frac{1}{n^2} \boldsymbol{1}^T(\sigma_w^2 (\sigma_x^2 + \mu_x^2) \mathbb{I} + \mu_w^2\sigma_x^2 \mathbb{J})\boldsymbol{1}\)

\(\text{Var}(\hat{\mu}) = \frac{1}{n^2} \mu_w^2\sigma_x^2\boldsymbol{1}^T \mathbb{J}\boldsymbol{1} \quad\quad \text{Leading order n}\)

\(\text{Var}(\hat{\mu}) = \mu_w^2\sigma_x^2\)

\(\text{Var}(\hat{\mu}) = \mu_w^2\sigma_x^2\)

Matrix cookbook eq. (379):

\(\text{Var}(\mathbf{x}^T\mathbf{A}\mathbf{x}) = \text{Tr}[\mathbf{A}\Sigma(\mathbf{A}+\mathbf{A}^T)\Sigma] + \boldsymbol{\mu}^T(\mathbf{A}+\mathbf{A}^T)\Sigma(\mathbf{A}+\mathbf{A}^T)\boldsymbol{\mu}\)


\(\text{Var}(\hat{\sigma}^2) = \frac{2}{n^2}\text{Tr}[(\mathbb{I} - \frac{1}{n} \mathbb{J})\Sigma_y(\mathbb{I} - \frac{1}{n} \mathbb{J})\Sigma_y] + \frac{4}{n^2}\boldsymbol{\mu}_y^T(\mathbb{I} - \frac{1}{n} \mathbb{J})\Sigma_y(\mathbb{I} - \frac{1}{n} \mathbb{J})\boldsymbol{\mu}_y\)

\(\text{Var}(\hat{\sigma}^2)\):


\(\text{Tr}[(\mathbb{I} - \frac{1}{n} \mathbb{J})\Sigma_y(\mathbb{I} - \frac{1}{n} \mathbb{J})\Sigma_y]\)

\(= \text{Tr}[((\mathbb{I} - \frac{1}{n} \mathbb{J})\Sigma_y)^2]\)

\(= \text{Tr}[((\mathbb{I} - \frac{1}{n} \mathbb{J})(\sigma_w^2 (\sigma_x^2 + \mu_x^2) \mathbb{I} + \mu_w^2\sigma_x^2 \mathbb{J}))^2]\)

\(= \text{Tr}[(\sigma_w^2 (\sigma_x^2 + \mu_x^2) \mathbb{I} + \mu_w^2\sigma_x^2 \mathbb{J} - \frac{1}{n}\mu_w^2\sigma_x^2 \mathbb{J}^2)^2]\quad\quad\text{Leading order in n}\)

\(= \text{Tr}[(\sigma_w^2 (\sigma_x^2 + \mu_x^2) \mathbb{I})^2]\)

\(= \sigma_w^4 (\sigma_x^2 + \mu_x^2)^2\text{Tr}[\mathbb{I}]\)

\(= n \sigma_w^4 (\sigma_x^2 + \mu_x^2)^2\)

\(\text{Var}(\hat{\sigma}^2)\):


\(\frac{4}{n^2}\boldsymbol{\mu}_y^T(\mathbb{I} - \frac{1}{n} \mathbb{J})\Sigma_y(\mathbb{I} - \frac{1}{n} \mathbb{J})\boldsymbol{\mu}_y\)


Since \(\boldsymbol{\mu}_y \propto \boldsymbol{1}\):

\((\mathbb{I} - \frac{1}{n} \mathbb{J})\boldsymbol{\mu}_y=\boldsymbol{0}\)

\(\text{Var}(\hat{\sigma}^2)\):


\(\text{Var}(\hat{\sigma}^2) = \frac{2}{n^2}\text{Tr}[(\mathbb{I} - \frac{1}{n} \mathbb{J})\Sigma_y(\mathbb{I} - \frac{1}{n} \mathbb{J})\Sigma_y]\)

\(\text{Var}(\hat{\sigma}^2) = \frac{2}{n}\sigma_w^4 (\sigma_x^2 + \mu_x^2)^2\)

\(\text{Var}(\hat{\sigma}^2)\):

\(\text{Var}(\hat{\sigma}^2) = \frac{2}{n}\sigma_w^4 (\sigma_x^2 + \mu_x^2)^2\)

Numerical Checks

import numpy as np

num_iters = 100

N = 1000

u_w = 7
u_x = 8
s_w = 3
s_x = 9

f_u = np.sqrt(N) * u_w * u_x
f_v = s_w**2 * (s_x**2 + u_x**2)
f_v_u = u_w**2 * s_x**2
f_v_v = 2 * f_v**2 / N

e_list = []
v_list = []

for _ in range(num_iters):

    W = np.random.normal(loc=u_w, scale=s_w, size=(N, N))
    x = np.random.normal(loc=u_x, scale=s_x, size=(N,))

    y = W @ x / (np.sqrt(N))
    u = np.sum(y) / N
    v = np.sum((y - u) ** 2) / N

    e_list.append(u)
    v_list.append(v)

e_u = sum(e_list) / num_iters
e_v = sum(v_list) / num_iters
v_u = sum([(e - e_u)**2 for e in e_list]) / num_iters
v_v = sum([(v - e_v)**2 for v in v_list]) / num_iters

print(f"Empirical mean: {e_u}")
print(f"Formula mean: {f_u}")

print(f"Empirical variance: {e_v}")
print(f"Formula variance: {f_v}")

print(f"Empirical variance of sample mean: {v_u}")
print(f"Formula variance of sample mean: {f_v_u}")

print(f"Empirical variance of sample variance: {v_v}")
print(f"Formula variance of sample variance: {f_v_v}")
